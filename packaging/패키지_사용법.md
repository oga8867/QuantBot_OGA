# 패키징 도구 (다른 PC로 공유하기)

이 폴더는 퀀트봇을 **다른 환경으로 옮기기 위한 도구**입니다.
봇 실행과는 무관하며, "공유용 zip 만들기"에만 씁니다.

## 구성
| 파일 | 역할 |
|---|---|
| `pack.bat` | 더블클릭하면 공유용 zip을 만듭니다 (pack.ps1 호출) |
| `pack.ps1` | 실제 압축 로직 (PowerShell). 프로젝트 루트 기준으로 동작 |
| `패키지_사용법.md` | 이 문서 |

> `pack.ps1`은 `packaging/` 안에 있어도 **자동으로 상위(프로젝트 루트)로 이동**해
> 프로젝트 전체를 올바르게 압축합니다. 폴더 안에 있어도 정상 작동합니다.

## 사용법
1. `packaging/pack.bat` **더블클릭**
2. 모드 선택:
   - **[1] 코드만** — API키·DB·보고서 제외. 새 PC에서 처음부터 시작 (~1MB)
   - **[2] 코드 + 거래기록** ★권장 — DB 포함, **API키 자동 제거**(user_settings.json·.env·KIS 토큰 스크럽)
   - **[3] 전부 포함** — API키 평문 포함 (분실 주의)
3. 프로젝트 루트에 `quant-bot_날짜시각.zip` 생성됨

## 받은 zip을 새 PC에서 쓰기
1. zip 압축 해제
2. **`setup.bat` 더블클릭** (파이썬·가상환경·라이브러리 자동 설치 + 대시보드 실행)
3. 대시보드 설정 탭에서 API 키 입력 (모드 1·2로 받았다면)

## 안전장치
- **모드 2**는 `user_settings.json`의 KIS/디스코드/텔레그램 키를 **자동으로 빈 값**으로 바꾸고,
  `data/kis_token_*.json`(유효 토큰)도 제거한 뒤 압축합니다. → 실수로 키가 새지 않습니다.
- 원본 파일은 절대 건드리지 않습니다 (임시 폴더에 복사 후 스크럽).
- 생성된 zip(`quant-bot_*.zip`)은 `.gitignore`로 git에 안 올라갑니다.

> ⚠️ 모드 3 zip은 **API 키가 평문**입니다. 신뢰하는 본인 PC로만 옮기세요.
